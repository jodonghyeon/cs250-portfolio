Project Hello Quad

3D Graphics

Project Overview

In this assignment, I implemented a setup the foundational rendering features to display and animate a quad using OpenGL.

Tasks Completed

  • Implemented classes to encapsulate OpenGL Textures, VertexBuffers, IndexBuffers, and VertexArrays.
  • Created vertex shader and fragment shader to display a quad mesh and animate based on time, and mouse position.

Reflection

During development, I learned in detail about the usage of OpenGL functions used for rendering and how they work. Basically, I learned that in order to access a buffer, I have to bind it first. And there was a problem where textures were not drawn, and while solving that problem, I learned that in order to properly bind and use textures in OpenGL, I have to activate the 'texture unit'.
